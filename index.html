Rivet TimeSince Formatter

A rivet's formatter for displaying time in facebook format: "6 minutes ago", "2 days ago", "1 week ago"

Download source - Demo page

Example

Filtered text / Result

08/04/2016 15.44 is time ago

Usage

<html>
    <body>
        <div id="app">
            <p>
                <!-- use the formatter -->
                08/04/2016 15.44 is <span rv-text="now | timesince">time ago</span>
            </p>
        </div>

        <!-- include rivets -->
        <script src="https://cdnjs.cloudflare.com/ajax/libs/rivets/0.8.1/rivets.bundled.min.js"></script>

        <!-- include the formatter -->
        <script src="timesince.formatter.js"></script>

        <!-- rivets init -->
        <script>

            rivets.bind( document.getElementById('app'), {now: new Date(1460123067142)})
        </script>
    </body>

</html>
